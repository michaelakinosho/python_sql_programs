Required Task Detail

Fields that need to be captured within tasks

Three types of task have been defined:

    Types of Tasks:
  1. Todos
  2. Meetings
  3. Phone Calls

Todos Task type is the basic task type, and requires the following fields:

  1. Title
  2. Description
  3. Due Date and Time are required e.g. July 1, 2022 11:50 pm
  4. Priority; use a dropdown
    1. Low
    2. Medium
    3. High
  5. Status
    1. Unstarted
    2. In Progress
    3. Completed
    4. Discharged
        5. Question to customer and team lead; Will this flag be set by clicking the task as archived or user has to set this value
  6. Assignees
      7. Assignees should be listed in two textboxes:
    1. Primary Assignee (One person only)
    2. Secondary Assignees
    3. Relationship of Assignees to Tasks Needs to be handled in a Many-To-Many Table
  7. Contact Name
  8. Contact URL

Meeting (In-person, face-to-face, non-virtual) Task Type will include all of the fields of the Todos Task Type and the following fields:

  1. Address of Meeting Location
  2. City, State, and Zipcode

Phone call (Zoom, virtual, remote) Task Type will include all of the fields of the Todos Task Type and the following fields:

  1. Phone Number
  2. Conference Call Number
  3. Video Conference URL
